# R Code

This repository contains the code used to perform the analyses described in : 

Magali Miserocchi, Pauline De Meyer et al., Tgf-β signaling positively regulates microglia development in the zebrafish embryo, 2026
 

## Requirements 

The following R libraries need to be installed : Seurat, SeuratObject, sctransform, scDblFinder
